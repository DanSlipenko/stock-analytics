# Stock Analytics

This is a standard Next.js app using the App Router.

## Edit the App

Edit source files in `src`, not files inside `.next`.

- Home page: `src/app/page.tsx`
- Alerts page: `src/app/alerts/page.tsx`
- Campaigns page: `src/app/campaigns/page.tsx`
- Watchlist page: `src/app/watchlist/page.tsx`
- Shared components: `src/components`
- API routes: `src/app/api`

The `.next` folder is generated by `next dev` and can be deleted safely. Next.js will recreate it when the dev server runs.

## Run Locally

```bash
npm install
npm run dev
```
